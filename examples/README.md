# Examples

These pages are conformance fixtures for the infographic CSS library.

Layering:

- Contract: skill/spec documents define the allowed surface
- Implementation: `infographic.css` renders the contract
- Fixtures: the HTML files here prove the contract and implementation line up

Fixtures:

- `shell/shell.html` - canonical page shell fixture
- `families/process.html` - process family
- `families/comparison.html` - comparison family
- `families/structure.html` - structure family
- `families/analysis.html` - analysis family
- `families/data.html` - data family
- `families/code.html` - code family

Each fixture references `../../infographic.css`.
