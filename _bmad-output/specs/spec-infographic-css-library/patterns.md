# Pattern Catalog

## Process Pattern

Use for ordered progression, timelines, roadmaps, pipelines, and checklists.

Variants:

- `ig-process-variant-timeline`
- `ig-process-variant-roadmap`
- `ig-process-variant-pipeline`
- `ig-process-variant-checklist`

Required roles:

- `ig-process-pattern`
- `ig-process-header`
- `ig-process-kicker`
- `ig-process-title`
- `ig-process-summary`
- `ig-process-steps` or `ig-process-pipeline`
- `ig-process-step` or `ig-process-pipeline-stage`
